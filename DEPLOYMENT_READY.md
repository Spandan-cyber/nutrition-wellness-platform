## 🎯 The 5 Essential Steps

### 1. Create GitHub Actions Workflow (5 min)
**File:** `.github/workflows/deploy.yml`
- Go to GitHub web UI
- Create new file in `.github/workflows/`
- Copy workflow content from setup guide
- Commit

### 2. Enable GitHub Pages (1 min)
**Location:** Repository Settings → Pages
- Select `gh-pages` branch
- Select `/ (root)` folder
- Save

### 3. Update Google OAuth (5 min)
**Location:** Google Cloud Console → Credentials
- Add: `https://Spandan-cyber.github.io` to origins
- Add: `https://Spandan-cyber.github.io/nutrition-wellness-platform/` to URIs
- Save

### 4. Trigger Deployment (2-3 min)
**Automatic:** GitHub Actions runs after workflow file creation
- Monitor in Actions tab
- Wait for build to complete

### 5. Test Your Site (5 min)
**Visit:** `https://Spandan-cyber.github.io/nutrition-wellness-platform/`
- Test all features
- Verify Google OAuth works

**Total Time: ~18 minutes**
